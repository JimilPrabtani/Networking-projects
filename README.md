# Networking & Network Security Projects

A collection of hands-on networking and network security projects covering a range of topics including routing, switching, VPNs, firewalls, and secure network design.

These projects are built using industry-standard tools and protocols, and are intended to demonstrate practical skills in configuring, securing, and troubleshooting network infrastructure.

## Topics Covered

- Network routing and switching
- Site-to-Site and Remote Access VPNs
- Firewall configuration and access control
- Network security protocols (IPSec, IKE, TLS)
- Network design and topology planning
